# ADR-004 — Laravel Convention First

Status: Accepted

Date: 2026-06-18

## Context

Tujuan utama adalah maintainability dan onboarding yang mudah bagi developer Laravel.

## Decision

Mengikuti konvensi Laravel sebanyak mungkin.

Menggunakan:

- Eloquent
- Form Request
- Policy
- Service Container

Tidak mewajibkan:

- Repository Pattern
- DTO Pattern

## Consequences

Boilerplate lebih sedikit.

Developer Laravel dapat memahami codebase lebih cepat.
