# ADR-008 — Theme Engine Architecture

Status: Accepted

Date: 2026-06-18

## Context

Sekolah membutuhkan kemampuan mengganti tampilan website.

## Decision

Theme berbasis filesystem.

Theme aktif disimpan pada settings.

Tidak menggunakan visual builder.

## Consequences

Implementasi lebih sederhana.

Tidak mendukung drag-and-drop page builder.
