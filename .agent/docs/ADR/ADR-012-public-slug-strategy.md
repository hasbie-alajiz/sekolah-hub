# ADR-012 — Public Slug Strategy

Status: Accepted

Date: 2026-06-18

## Context

URL publik harus stabil dan mudah dibaca.

## Decision

Seluruh entitas publik menggunakan slug.

Slug bersifat global unique.

## Consequences

Tidak ada konflik URL antar modul publik.

Routing publik menjadi lebih sederhana.
